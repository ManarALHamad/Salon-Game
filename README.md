# 💇🏻‍♀️ Salon Game

## Overview
Salon Game is a fun and interactive browser game where your goal is to dry a customer's hair before time runs out. You have **20 seconds** to completely dry all sections of the hair using the hair dryer.

## 🎮 How to Play
1. Click the **Start** button.
2. Drag the **hair dryer** using your mouse.
3. Point the hair dryer at each section of the customer's hair.
4. Keep the dryer on **each hair section for 5 seconds** until it changes color to light, indicating it is dry.
5. Dry **all hair sections within 20 seconds** to win.

## 🏆 Win Condition
- Successfully dry all hair sections in 20 seconds or less.

## ❌ Lose Condition
- If the timer reaches 20 seconds before all hair sections are dry, you lose.

## ✨ Features
- Drag-and-drop hair dryer.
- Interactive hair drying mechanics.
- Countdown timer.
- Win and lose screens.
- Replay button to start a new game.
